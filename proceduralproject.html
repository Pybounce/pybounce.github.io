Procedural Project

Procedurally generated endless terrain, with highly customizable user settings to change the look of the scenery, and a jet to fly around in.

Features

Chunk System - Dynamically adds and removes chunks smoothly, at a given render distance.

Customisable Settings - The player can change the colour of all terrain types and the skybox in real time during a game.

Main Menu - This dynamic main menu backdrop generates new randomised terrain each time it's opened.

Jet Controller - A semi-arcade movement style jet with fully animated flaps, used to traverse the mountains.
